# Sovereign-CDS-vs-Asset-Classes-Hedge-and-Safe-Haven-Research

Multiscale Financial Risk Analysis (MODWT)

A wavelet-based framework to analyze financial risk, volatility, and hedging behavior across different time horizons.

🚀 Why this matters

Financial markets behave differently in the short term vs long term.
This project captures those differences by breaking time series into multiple scales, helping uncover hidden risk patterns and more effective hedging strategies.

⚙️ What’s inside
Time series decomposition using MODWT
Scale-wise volatility and correlation analysis
Multiscale hedge ratio estimation
Hedging effectiveness evaluation

📁 Structure
data/        → input datasets  
notebooks/   → analysis & visualizations  
src/         → core logic  
results/     → outputs & plots  
utils/       → helper functions  

📈 Key Insights
Risk is not uniform across time horizons
Short-term movements ≠ long-term trends
Multiscale analysis leads to better risk and hedging decisions

🛠️ Stack
Python • NumPy • Pandas • PyWavelets • Jupyter

🎯 Use Cases
Market & Credit Risk
Portfolio Hedging
Quantitative Research

💡 Takeaway
Move beyond traditional models — understand markets through a multiscale lens.
